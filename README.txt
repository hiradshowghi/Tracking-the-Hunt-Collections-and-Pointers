This program exists to print and track the list of rooms as well as ghosts that might be in it. It also has the ability to read and tell you the most/least likely ghosts and a feature to sort the likelihood of all ghosts. 

Files included in zip:

README.txt 
Room.c
main.c
building.c
ghost.c
defs.h

Instructions to compile:

Go to the directory containing files and type: gcc -o a4 main.c building.c ghost.c room.c

To run after compiling: ./a4 or whatever you put instead of a4

Once running there will be 5 options to choose from, enter the number key associated with what you want to do or 0 to exit.
